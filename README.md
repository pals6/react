# React Projects

This repository contains multiple React + Vite projects by **Pallavi Bichpuriya**.

## Projects

### 1. Community Board

- Folder: `communityboard/`
- Description: A campus food truck board with responsive cards and menu links.
- Details: See `communityboard/README.md`

### 2. AI Interview Flashcards

- Folder: `ai-interview-flashcards/`
- Description: A flashcard app for practicing AI, ML, and CS interview concepts.
- Details: See `ai-interview-flashcards/README.md`

### 3. Timetabled

- Folder: `timetabled/`
- Description: A 7-day California trip planner/calendar app.
- Details: See `timetabled/README.md`

### 4. Veni Vici

- Folder: `veni-vici/`
- Description: A dog discovery app that fetches random breeds from The Dog API and lets users ban attributes from future results.
- Details: See `veni-vici/README.md`

### 5. Barista App

- Folder: `barista-app/`
- Description: A coffee order guessing game where users build drink recipes and check their answers against randomized barista orders.
- Details: See `barista-app/README.md`

## Run a Project Locally

Each project is standalone. Run commands inside the specific project folder:

```bash
cd <project-folder>
npm install
npm run dev
```

Examples:

```bash
cd veni-vici
npm install
npm run dev
```
